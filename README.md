# Balance Scale

An interactive support and API troubleshooting demo tool built with HTML, CSS, and JavaScript.

## Overview

Balance Scale is a visual comparison tool.

The project uses an animated hanging scale interface to visually represent impact of an employee in a more engaging and intuitive way.

This project was created as a portfolio piece to demonstrate:

* Frontend development skills
* Technical troubleshooting logic
* API integration concepts
* UX thinking for support workflows
* Analytical problem-solving

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/balance-scale.git
```

Open the project folder:

```bash
cd balance-scale
```

Run locally by opening:

```bash
index.html
```
